PROJECT 03 Service Agent

LLM 에너지 분석 챗봇 Solution Summary Agent

프로젝트 개요

비개발자도 SQL 작성 없이 자연어로 복잡한 건물 에너지 데이터를 조회·분석할 수 있는 LLM 기반 멀티턴 대화형 에이전트입니다. LangGraph를 통해 사고 과정을 구조화하여 데이터 접근성을 혁신했습니다.

주요 역할 및 성과

  • LangGraph 아키텍처: 상태(State) 기반 멀티턴 대화 제어 및 에이전트 루프 설계로 문맥 유지 능력 강화
  • Multi-Tool Orchestration: SQL Toolkit(DB), 예측 모델 API, 매뉴얼 검색(RAG)의 유기적 결합
  • System Prompt Engineering: 도메인 특화 쿼리 생성 규칙 및 할루시네이션 방지 로직 구현
  • Data Accessibility: "지난달 A동 전력 비교해줘" 등 자연어 질의로 즉시 데이터 접근 환경 구축
