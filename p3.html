PROJECT 01 Production

건물 재실 예측 솔루션 Occupancy Assumption Service

프로젝트 개요

BEMS(건물에너지관리시스템)의 핵심 기능으로, 실시간 전력 사용 패턴과 조명 상태 데이터를 분석하여 현재 공간의 재실 여부(Occupancy)와 밀도를 예측하는 머신러닝 서비스입니다.

주요 역할 및 성과

  • 듀얼 알고리즘 아키텍처 설계: 설비 특성에 맞춰 HMM(전열)과 KMeans(전등) 모델 이원화 적용
  • 데이터 전처리 파이프라인: Hampel Filter를 활용한 이상치 제거 및 결측치 보간으로 데이터 품질 확보
  • BentoML 기반 API 배포: 모델 서빙 최적화를 통해 평균 응답시간 3~10초 이내 달성
  • 운영 모니터링 구축: Grafana/Prometheus 연동으로 False Positive 패턴 실시간 감지
